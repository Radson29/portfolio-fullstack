Car dealership
Aplikacja Car Dealership to kompleksowy system do zarządzania salonem samochodowym, obejmujący sprzedaż pojazdów, obsługę serwisową oraz zarządzanie pracą mechaników. Dzięki integracji z API CEPiK możliwe jest generowanie przykładowych danych pojazdów w celach edukacyjnych i prezentacyjnych. Projekt wykorzystuje Spring Boot, PostgreSQL oraz Thymeleaf, zapewniając wygodne interfejsy dla klientów, sprzedawców i mechaników. Dodatkowo aplikacja posiada rozbudowane testy (TestContainers, WireMock, Rest Assured) i dokumentację API (OpenAPI/Swagger).
