Fullstack Web App
MediSync
MediSync to system zarządzania danymi medycznymi stworzony z myślą o placówkach zdrowia i zespołach operacyjnych. Umożliwia kompleksowe zarządzanie pacjentami, użytkownikami i ich rolami, salami operacyjnymi, zasobami, oceną przedoperacyjną oraz operacjami medycznymi. Backend oparty na Spring Boot udostępnia bezpieczne REST API z obsługą JWT oraz strukturą opartą na DTO. Frontend, zbudowany w Angularze, zapewnia dynamiczny i intuicyjny interfejs użytkownika.
